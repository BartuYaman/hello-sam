# hello-sam

A small, self-contained [AWS SAM](https://github.com/awslabs/serverless-application-model) hello-world. Uses make to automate and document everything.

# prerequisites

 * `$ aws` configured & running
 * `$ make`
 * `$ jq`
 * `$ npm install -g aws-sam-local` 

# development

* `$ make help`
